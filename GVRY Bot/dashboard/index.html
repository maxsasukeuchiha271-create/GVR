Quick Reference

Config saves (Bot Settings / Tickets / Vanity / Embeds / Emojis) take effect immediately — no restart needed.
In embed text: {user} {required} {frp_speeds} {peacetime} {leo} {notes} {start_time} {end_time} are runtime values.
Emoji vars like {arrow} {stars} expand automatically — define them in the Emojis tab.
Esc closes modals  ·  Ctrl S saves the current config page
